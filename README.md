# 📊 Análise Financeira de Restaurante

## 🎯 Objetivo
Realizar análise de dados financeiros de um restaurante, com foco em identificar padrões, gerar insights e apoiar a tomada de decisão orientada a dados (Data-Driven Decision Making).

---

## 🧠 Tecnologias Utilizadas
- Python  
- Pandas  
- Matplotlib  
- Google Colab  

---

## 📊 Etapas da Análise (Data Analysis)
- Coleta e estruturação de dados financeiros  
- Limpeza e tratamento de dados (Data Cleaning)  
- Análise exploratória de dados (EDA)  
- Agrupamento e agregação de dados (GroupBy)  
- Visualização de dados (Data Visualization)  

---

## 📈 Principais Métricas (KPIs)
- Receita total  
- Despesa total  
- Lucro líquido  
- Receita por categoria  
- Despesa por categoria  

---

## 🔍 Principais Insights
- As receitas estão concentradas principalmente em vendas no balcão e delivery  
- Despesas com salários representam a maior parcela dos custos operacionais  
- O lucro é diretamente impactado pelo volume de vendas diárias  
- Identificação de categorias com maior impacto financeiro no resultado  

---

## 🚀 Como Executar
1. Acesse o notebook no Google Colab  
2. Execute todas as células  
3. Analise os resultados e gráficos gerados
